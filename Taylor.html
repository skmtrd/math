関数の振る舞いに関する説明

概要

関数 \( f(x) \) が少なくとも \( n \) 回微分可能であり、次の条件が成り立つとします：

  • \( f^{(k)}(a) = 0 \)（\( k = 0, 1, \ldots, n-1 \)）
  • \( f^{(n)}(a) \neq 0 \)

このとき、\( n \) の偶奇によって \( f(x) \) の \( x = a \) 近傍での振る舞いが異なります。

\( n \) が偶数の場合

主要項は以下のように表されます：

\[ f(x) = f(a) + \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n+1}) \]

- \( f^{(n)}(a) > 0 \): \( f(x) \) は \( x = a \) を中心にして極小値を持ちます。
- \( f^{(n)}(a) < 0 \): \( f(x) \) は \( x = a \) を中心にして極大値を持ちます。

結論として、\( n \) が偶数の場合、\( f(x) \) は \( x = a \) で明確な極値を持ちます。

\( n \) が奇数の場合

主要項は以下のように表されます：

\[ f(x) = f(a) + \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n+1}) \]

- \( f^{(n)}(a) > 0 \): \( f(x) \) は \( x = a \) を通過し上昇します。
- \( f^{(n)}(a) < 0 \): \( f(x) \) は \( x = a \) を通過し下降します。

結論として、\( n \) が奇数の場合、\( f(x) \) は \( x = a \) で極値を持たず、単調に変化します。
